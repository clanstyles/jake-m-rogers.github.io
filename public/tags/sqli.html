SQLMap Tamper Scripts Update

published on July 04, 2016
So my last post from three years ago was pretty well received, with the rate at which SQLMap is developed it is definitely behind.r, Here’s an updated graph of all tamper scripts included with the latest version (as of time of writing, 04/July/2016). I will be putting something together, possibly as a commit to the SQLMap project to aid with these. Will update once done so stay tuned.
